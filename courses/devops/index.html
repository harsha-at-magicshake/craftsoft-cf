Course Curriculum

Build the DevOps mindset, automate deployments, and work with production-grade systems.

Beginner - DevOps & Linux Foundations

Goal: Build the mindset and base skills required for DevOps.

1 DevOps & CI/CD Culture

  • What is DevOps & CI/CD culture
  • Linux fundamentals (commands, files, permissions)

2 Git & Networking

  • Git & GitHub workflows
  • Basics of networking (IP, ports, protocols)
  • Introduction to cloud & virtualization

Beginner Outcomes

  • Understand DevOps lifecycle
  • Core tools used in real teams

Intermediate - Containers & CI/CD

Goal: Automate builds and deployments.

1 CI/CD & Docker

  • Jenkins pipelines (CI/CD)
  • Docker fundamentals & images
  • Docker Compose

2 Kubernetes Basics

  • Kubernetes basics (pods, services, deployments)
  • Infrastructure automation concepts

Intermediate Outcomes

  • Containerize applications
  • Automate deployment pipelines

Advanced - Scalable DevOps Systems

Goal: Work with production-grade DevOps setups.

1 Advanced Kubernetes

  • Advanced Kubernetes (scaling, configs, secrets)
  • Monitoring & logging basics
  • CI/CD pipeline optimization

2 Cloud & Career

  • Cloud-based deployments
  • Real-world DevOps project
  • Interview & troubleshooting prep

Advanced Outcomes

  • Job-ready DevOps Engineer
  • Production understanding
